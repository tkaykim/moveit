# 학원 Admin 기능 소개 및 가이드 (메일용)

**수신:** 학원 Admin 사용자  
**목적:** MOVEIT 학원 관리 시스템의 각 메뉴 기능 안내 및 활용 가이드

---

안녕하세요.

MOVEIT 학원 관리(Admin) 시스템을 이용해 주셔서 감사합니다.  
댄스학원 등 실제 학원 운영 경험을 바탕으로 설계한 기능들을 정리해 두었습니다.  
아래 내용을 참고하시어 일상 업무에 활용해 주시기 바랍니다.

---

## 1. 대시보드

- **역할:** 관리 화면의 시작점입니다.
- **기능:**
  - **클래스 관리**, **스케줄 관리**, **신청인원 관리**, **수강권 관리**로 바로 이동하는 카드 버튼
  - **오늘의 수업 일정** 요약
- **활용:** 매일 접속 시 오늘 수업을 확인하고, 필요한 메뉴로 빠르게 이동할 때 사용합니다.

---

## 2. 학생 관리

- **역할:** 학원에 등록된 회원(학생)을 조회·등록·관리합니다.
- **기능:**
  - 학생 목록 조회 및 검색(이름, 연락처 등)
  - **학생 등록**: 새 회원 추가
  - **학생 상세**: 보유 수강권, 수업 예약/출석 이력 확인
- **활용:** 회원 가입·문의 시 조회, 수강권·출석 현황 확인 시 사용합니다.

---

## 3. 클래스(반) 관리

- **역할:** “어떤 수업을 운영할지”를 정의하는 **수업 마스터**를 만드는 메뉴입니다.  
  실제 **요일·시간에 배치**하는 것은 **스케줄 관리**에서 합니다.

### 3-1. 클래스(반)란?

- **클래스(반)** = 장르, 난이도, 강사, 정원 등이 정해진 **수업 종류**입니다.  
  예: “월요일 초급 힙합”, “수요일 중급 치어오”처럼 **반 단위**로 관리하는 개념입니다.

### 3-2. 수강권 연결 기능 (중요)

- 각 클래스(반)에 **어떤 수강권으로 신청·출석할 수 있는지**를 연결합니다.
- **기간제 수강권**, **쿠폰제(횟수제) 수강권**, **워크샵(특강) 수강권**을 클래스별로 허용/비허용 설정 가능합니다.
- **정규 전용**, **팝업만**, **둘 다 허용** 등 학원 정책에 맞게 설정할 수 있어,  
  댄스학원에서 “이 반은 정규만”, “이 반은 드롭인/팝업 가능” 같은 운영을 그대로 반영할 수 있습니다.

### 3-3. 클래스 유형

- **Regular(정규)**: 반복 운영하는 정규 수업
- **Popup(팝업)**: 단발성·특별 수업
- **Workshop(워크샵)**: 워크샵 형태 수업

클래스 생성·수정 시 **유형**과 **연결할 수강권 종류**를 함께 설정합니다.

---

## 4. 스케줄 관리

- **역할:** 이미 만든 **클래스(반)**를 **날짜·시간·장소**에 배치하여 “실제 수업 일정”을 만드는 메뉴입니다.

### 4-1. 수업을 만들고 일정을 배치하는 순서 (필수)

1. **클래스(반) 먼저 만들기**  
   → **클래스(반) 관리**에서 수업 종류(장르, 난이도, 강사, 수강권 연결 등)를 등록합니다.
2. **스케줄에 배치하기**  
   → **스케줄 관리**에서 해당 클래스를 선택한 뒤,  
   - **반복 스케줄**: 매주 요일·시간대로 일괄 생성  
   - **특정 일자**: 원하는 날짜에만 세션 추가  
   로 “언제, 어디서, 누가 가르치는지”까지 정해진 **세션(일정)**을 만듭니다.

즉, **클래스 = 수업 종류**, **스케줄(세션) = 그 수업이 열리는 구체적인 날·시간**입니다.  
클래스를 먼저 만들어 두어야 스케줄 생성 시 선택할 수 있습니다.

### 4-2. 스케줄 관리에서 할 수 있는 일

- **달력 보기**: 월별로 세션(수업 일정) 확인
- **스케줄 생성**: 반복 또는 단일 세션 생성 (클래스·홀·강사·시간 지정)
- **세션 클릭**: 해당 일정의 상세 보기·수정·취소·삭제
- **홀(장소) 필터**: 여러 연습실이 있을 때 홀별로 일정만 보기

### 4-3. 스케줄별 링크 생성 기능 (결제/신청 링크)

- 스케줄 관리 달력에서 **세션(수업 일정)**을 클릭하면 **세션 상세** 창이 열립니다.
- 여기서 **「결제 링크 복사」** 버튼을 누르면,  
  **해당 세션(수업 일정)에 대한 신청·결제 페이지 링크**가 클립보드에 복사됩니다.
- 이 링크를 SNS, 문자, 메일 등으로 회원에게 보내면,  
  회원이 해당 **그 날, 그 시간 수업**으로 바로 신청·결제할 수 있습니다.
- **세션(일정) 단위로 링크가 나뉘어** 있어, “이번 주 수요일 7시 수업만 링크로 받기” 같은 운영이 가능합니다.

---

## 5. 신청인원 관리

- **역할:** 특정 **스케줄(세션)** 또는 **날짜** 기준으로, 누가 어떤 수업에 신청했는지 관리합니다.
- **기능:**
  - 스케줄/날짜 선택 후 **예약(신청) 목록** 조회
  - 상태: **확정**, **대기**, **취소**, **출석 완료**
  - 관리자가 회원을 대신해 **신청 추가**(Admin에서 직접 등록)
- **활용:** 전화/현장 접수 후 시스템에 반영하거나, 출석 처리·취소 처리 시 사용합니다.

---

## 6. 수강권/상품 탭

- **역할:** 학원에서 판매하는 **수강권·상품**을 등록·수정·판매 중단 관리합니다.
- **기능:**
  - 수강권 생성: 이름, 유형(정규/팝업/워크샵), 가격, 유효일 등
  - **할인 정책** 별도 등록·관리
  - 수강권별 **클래스 연결**은 **클래스(반) 관리**에서 설정
- **활용:** 새 상품 출시, 가격·유효기간 변경, 할인 이벤트 적용 시 사용합니다.

---

## 7. 업무 수업일지 탭

- **역할:** **날짜별**로 그날 있었던 수업·운영 내용을 기록하고 조회합니다.
- **기능:**
  - 날짜 선택 시 해당 날짜의 **스케줄(수업 일정)** 목록
  - **수업일지** 작성·수정 (수업 내용, 특이사항 등)
  - **운영 메모** (해당 날짜 공통 메모)
- **활용:** 일일 업무 로그, 강사·회원 피드백 정리, 나중에 일정·수업 내용 검색 시 사용합니다.

---

## 8. 강사 관리 탭

- **역할:** 학원에 소속된 **강사**를 등록·조회·관리합니다.
- **기능:**
  - 강사 목록, 소속 학원 연결
  - 강사 검색·추가 (기존 강사 풀에서 연결)
- **활용:** 클래스(반) 또는 스케줄에 강사를 지정할 때, 여기서 등록된 강사가 선택됩니다.

---

## 9. 상담 관리 탭

- **역할:** **상담 신청/예약**을 한 건 단위로 관리합니다.
- **기능:**
  - 상담 목록: 신청자, 연락처, 주제, 희망 일시 등
  - 상태: **신규**, **예약됨**, **완료**, **취소**
  - 담당자 지정, 일정 조율 후 상태 변경
- **활용:** 입회·재등록 상담, 이벤트 상담 등을 한곳에서 처리할 때 사용합니다.

---

## 10. 매출·정산 관리 탭

- **역할:** 학원의 **매출**과 **정산** 현황을 확인합니다.
- **기능:**
  - **이번 달 매출** (결제 완료 기준)
  - **미정산 금액** (강사 정산 대기 등)
  - **순이익** (매출 − 미정산)
  - **최근 결제 내역** (회원, 수강권, 금액 등)
- **활용:** 일/월 매출 점검, 강사 정산 준비, 정산 완료 후 미정산 금액 확인 시 사용합니다.

---

## 요약: 클래스 ↔ 스케줄 ↔ 링크

| 단계 | 메뉴 | 하는 일 |
|------|------|--------|
| 1 | **클래스(반) 관리** | 수업 종류(반) 정의 + **수강권 연결** (정규/팝업/워크샵 허용 설정) |
| 2 | **스케줄 관리** | 그 클래스를 **날짜·시간·장소**에 배치 → **세션(일정)** 생성 |
| 3 | **스케줄 관리** (세션 클릭) | **「결제 링크 복사」** → 해당 세션만의 **신청·결제 링크** 발급 |

---

## 추후 제공 예정 기능 안내

- **기간권 연장**: 수강권 유효기간 연장 기능이 추가될 예정입니다.
- **기간권 일시정지**: 부상·휴가 등으로 수강을 잠시 멈출 때, 기간권을 일시정지하는 기능도 구현 예정입니다.

위 기능이 반영되면 별도로 안내드리겠습니다.
