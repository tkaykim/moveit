# 12. 매출/정산

## 1. 개요

결제 로그, 매출 통계, 수강권 판매를 관리합니다.

**경로**: `/academy-admin/{학원ID}/revenue`

---

## 2. 탭 구성

### 2-1. 매출 통계

- **이번 달 매출**: 해당 학원의 `revenue_transactions` 합계
- **미정산 금액**: 강사 정산 대기 금액 (구현 시)
- **순이익**: 매출 - 미정산

### 2-2. 결제 로그

- **최근 결제 내역** 테이블
- 날짜, 회원, 상품명, 원가, 할인, 최종 금액, 결제 방법
- `revenue_transactions` 기반

### 2-3. 수강권 판매 (탭 또는 별도 진입)

- **수강권 간편결제** (헤더 버튼) 또는 매출 페이지 내 탭으로 이동
- **학생 선택** → **수강권 선택** → **할인 적용** → **결제 확정**
- 결제 완료 시:
  - `user_tickets` 생성
  - `revenue_transactions` 기록
  - `academy_students` 자동 등록

---

## 3. 수강권 판매 플로우

1. **학생 검색**: 이름·연락처로 회원 선택
2. **수강권 선택**: 기간제 / 쿠폰제 / 워크샵, 옵션(1회/5회 등) 선택
3. **할인**: 할인 정책 또는 직접 금액/비율 입력
4. **결제 요약** 확인 후 **결제 완료**

---

## 4. 결제 로그 상세

- **학생명**, **상품명**, **유형**(기간/횟수), **원가**, **할인**, **최종금액**, **할인 상세**
