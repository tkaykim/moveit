# 08. 수강권/상품

## 1. 개요

기간제·쿠폰제·워크샵 수강권과 할인 정책을 관리하는 화면입니다. 수강권은 클래스와 연결하여 "어떤 수업을 들을 수 있는지"를 정합니다.

**경로**: `/academy-admin/{학원ID}/products`

---

## 2. 수강권 유형

| 유형 | 특징 | 적합 클래스 |
|------|------|-------------|
| **기간제** | 기간 내 무제한 수강, 횟수 차감 없음 | Regular |
| **쿠폰제(횟수제)** | N회권, 사용 시 1회 차감 | Popup |
| **워크샵(특강)** | 특정 워크샵 전용 N회권 | Workshop |

---

## 3. 기간제 수강권

### 3-1. 설정 항목

- **수강권명**
- **가격**
- **기간**: 1개월 / 3개월 / 6개월 / 12개월 또는 직접 입력
- **수강 가능 클래스**: 이 수강권으로 들을 수 있는 Regular 클래스 선택
- **전체 선택**: 모든 Regular 클래스에서 사용 가능 (일반 수강권)

### 3-2. 동작

- 구매 시 `start_date` ~ `expiry_date` 설정
- 기간 내 해당 클래스 수업은 **무제한** 수강 가능
- 사용할 때마다 횟수 차감 없음

---

## 4. 쿠폰제(횟수제) 수강권

### 4-1. 설정 항목

- **수강권명**
- **횟수권 옵션**: 1회/3회/5회/10회 등 옵션별 **가격**, **유효기간(일)** 설정
- 예: "취미반 쿠폰-1회 5,000원 / 5회 20,000원"

### 4-2. 동작

- 구매 시 선택한 옵션에 따라 `remaining_count`, `expiry_date` 설정
- 수업 신청 시 1회 차감
- 잔여 0이 되면 사용 완료
- 유효기간 만료 시 미사용 분은 소멸

---

## 5. 워크샵(특강) 수강권

### 5-1. 설정 항목

- **수강권명**
- **가격**
- **사용 가능 횟수**: 1회 / 3회 / 5회 / 10회 등
- **유효기간**: 30일 / 60일 / 90일 / 무제한 등

### 5-2. 동작

- **클래스(반) 관리**에서 Workshop 클래스를 만들 때 이 수강권을 연결
- 연결된 워크샵 수업에서만 사용 가능
- 사용 시 1회 차감

---

## 6. 할인 정책

- **할인 등록**: 정률(%) 또는 정액(원) 할인
- **수강권 판매 시** 또는 **관리자 판매 시** 적용 가능
- 유효기간, 학원 제한 등 설정 가능

---

## 7. 수강권 판매 (매출/정산)

- **매출/정산** 탭의 **수강권 판매** 영역에서:
  1. 회원 선택
  2. 수강권 선택
  3. 할인 적용
  4. 결제 확정
- 헤더의 **수강권 간편결제** 버튼으로도 해당 탭으로 이동 가능
