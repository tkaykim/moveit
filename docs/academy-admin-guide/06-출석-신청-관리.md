# 06. 출석/신청 관리

## 1. 개요

수업별 신청자 목록을 조회하고, 수동 등록·출석 처리·취소를 관리합니다.

**경로**: `/academy-admin/{학원ID}/enrollments`

---

## 2. 화면 구성

### 2-1. 날짜·수업 선택

1. **날짜 선택**: 특정 날짜 지정
2. **수업 선택**: 해당 날짜의 수업 목록에서 선택
3. 선택한 수업의 **신청자 목록**이 표시됩니다.

### 2-2. schedule_id URL

- URL에 `?schedule_id=xxx` 가 있으면, 해당 스케줄이 자동으로 선택됩니다.
- 출석/신청 관리 내부 링크나 스케줄에서 "신청인원" 클릭 시 이 방식으로 이동합니다.

---

## 3. 신청자 목록

| 표시 항목 | 설명 |
|----------|------|
| **이름** | 회원명 또는 비회원명 |
| **연락처** | 전화번호 |
| **상태** | 확정 / 대기 / 취소 등 |
| **결제** | 수강권 사용 / 카드결제 / 현장결제 등 |

### 3-1. 상태

- **확정(CONFIRMED)**: 수업 참석 확정
- **대기(PENDING)**: 승인 대기
- **취소(CANCELLED)**: 예약 취소

---

## 4. 수동 등록

### 4-1. 회원 추가

1. **수강생 추가** 또는 **등록 추가** 버튼 클릭
2. **회원 검색**으로 기존 학생 선택
3. **수강권 선택**: 보유 수강권 중 사용할 수강권 선택
4. **등록** → 수강권 1회 차감 후 예약 생성

### 4-2. 비회원(게스트) 추가

1. **수강생 추가** 또는 **게스트 추가** 클릭
2. **이름, 연락처** 입력
3. **등록** → 비회원으로 예약 생성

---

## 5. 액션 메뉴

각 신청자 행의 **⋮** 메뉴에서:

| 액션 | 설명 |
|------|------|
| **출석 처리** | 출석 처리 (일지와 연동 가능) |
| **취소** | 예약 취소 (쿠폰제 시 횟수 복구 여부 선택 가능) |
| **상세** | 신청 상세 정보 |

---

## 6. 요약 정보

- **총 인원**: 해당 수업의 신청자 수
- **정원**: `max_students` 기준
- **확정/대기/취소** 인원 수
