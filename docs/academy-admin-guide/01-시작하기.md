# 01. 시작하기

## 1. 접속 방법

학원 관리자 페이지는 다음 주소로 접속합니다.

```
https://[도메인]/academy-admin/[학원ID]
```

- **학원ID**: 각 학원마다 고유 ID가 부여됩니다. 로그인 후 자동으로 해당 학원 페이지로 이동합니다.
- 예: `https://moveit.example.com/academy-admin/abc123-def456-...`

---

## 2. 로그인

### 2-1. 관리자 로그인

1. 관리자 페이지 접속 시 로그인 화면이 표시됩니다.
2. **이메일**과 **비밀번호**를 입력합니다.
3. **로그인** 버튼을 클릭합니다.

### 2-2. 권한 유형

| 권한 | 설명 | 접근 가능 메뉴 |
|------|------|----------------|
| **ACADEMY_OWNER** | 학원 소유자 | 해당 학원의 모든 메뉴, 설정 |
| **ACADEMY_STAFF** | 학원 직원 | 해당 학원 대부분 메뉴 (설정 제한 가능) |
| **SUPER_ADMIN** | 최고 관리자 | 모든 학원 접근, 전체 설정 |

- **설정** 메뉴는 ACADEMY_OWNER 또는 SUPER_ADMIN만 표시됩니다.

---

## 3. 화면 구성

### 3-1. 레이아웃

```
┌─────────────────────────────────────────────────────┐
│  [메뉴]  대시보드              [수강권판매] [알림] [프로필] │  ← 헤더
├──────────┬──────────────────────────────────────────┤
│          │                                          │
│  대시보드 │                                          │
│  학생 관리│            본문 영역                        │
│  클래스   │                                          │
│  스케줄   │                                          │
│  ...     │                                          │
│          │                                          │
└──────────┴──────────────────────────────────────────┘
   ↑ 사이드바
```

### 3-2. 헤더 기능

| 항목 | 설명 |
|------|------|
| **제목** | 현재 페이지에 맞는 제목 표시 |
| **수강권 간편결제** | 수강권 판매 탭으로 바로 이동 |
| **온보딩** | 처음 사용 시 화면별 안내 튜토리얼 실행 |
| **알림** | 알림 목록 (구현 여부는 배포 환경에 따름) |
| **프로필** | 로그아웃, 설정 등 |

### 3-3. 사이드바

- **PC**: 왼쪽에 고정 표시
- **모바일**: 햄버거 메뉴(≡)를 누르면 슬라이드로 열림

---

## 4. 메뉴 개요

### 4-1. 수업 관리

| 메뉴 | 경로 | 설명 |
|------|------|------|
| 대시보드 | `/academy-admin/{id}` | 홈, 오늘 수업, 빠른 이동 카드 |
| 학생 관리 | `.../students` | 회원 등록·검색, 보유 수강권 |
| 클래스(반) 관리 | `.../class-masters` | 수업 종류(Regular/Popup/Workshop) 정의 |
| 스케줄 관리 | `.../schedule` | 월별 달력, 반복 일정 등록 |
| 출석/신청 관리 | `.../enrollments` | 수업별 신청자, 수동 등록 |

### 4-2. 운영 관리

| 메뉴 | 경로 | 설명 |
|------|------|------|
| 연장/일시정지 관리 | `.../extension-requests` | 수강권 연장·일시정지 신청 처리 |
| 수강권/상품 | `.../products` | 기간제·쿠폰·워크샵 수강권, 할인 |
| 업무/수업 일지 | `.../logs` | 일별 수업 일지, 출석·결강 처리 |

### 4-3. 매출 및 설정

| 메뉴 | 경로 | 설명 |
|------|------|------|
| 강사 관리 | `.../instructors` | 강사 등록, 클래스 배정 |
| 상담 관리 | `.../consultations` | 상담 예약·상태 관리 |
| 매출/정산 | `.../revenue` | 결제 로그, 매출 통계, 수강권 판매 |
| 설정 | `.../settings` | 학원 정보, 온보딩, 연장 정책 등 |

### 4-4. 기타

| 메뉴 | 경로 | 설명 |
|------|------|------|
| 고장신고/개발요청 | `.../support` | 버그 신고, 기능 요청 |

---

## 5. 운영 흐름 (처음 설정 시)

학원을 처음 사용할 때 권장 순서입니다.

1. **클래스(반) 관리**  
   - Regular, Popup, Workshop 등 수업 종류 생성
2. **수강권/상품**  
   - 기간제·쿠폰·워크샵 수강권 생성, 클래스 연결
3. **강사 관리**  
   - 강사 등록
4. **스케줄 관리**  
   - 반복 일정 등록, 달력에 수업 배치
5. **학생 관리**  
   - 회원 등록 또는 수강권 판매로 자동 등록
6. **출석/신청 관리**  
   - 수업별 신청 확인, 수동 등록
