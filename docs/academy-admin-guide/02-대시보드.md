# 02. 대시보드

## 1. 개요

대시보드는 학원 관리자 페이지의 **홈 화면**입니다. 오늘 수업 요약과 자주 쓰는 메뉴로 빠르게 이동할 수 있습니다.

**경로**: `/academy-admin/{학원ID}`

---

## 2. 주요 관리 카드 (빠른 이동)

| 카드 | 이동 경로 | 설명 |
|------|----------|------|
| **클래스 관리** | 클래스(반) 관리 | Regular/Popup/Workshop 클래스 생성·수정 |
| **스케줄 관리** | 스케줄 관리 | 반복 일정 등록, 월별 달력 |
| **출석/신청 관리** | 출석/신청 관리 | 수업별 신청자 확인, 수동 등록 |
| **수강권 관리** | 수강권/상품 | 수강권·할인 정책 관리 |

각 카드를 클릭하면 해당 메뉴로 바로 이동합니다.

---

## 3. 오늘 수업

- **오늘 날짜** 기준으로 해당 학원의 수업이 표시됩니다.
- 홀별로 **진행 중 / 예정 / 종료** 상태로 구분됩니다.
- 수업을 클릭하면 **출석/신청 관리** 또는 **일지** 화면으로 이동합니다.

### 3-1. 상태 구분

| 상태 | 설명 |
|------|------|
| **종료** | 수업 종료 시간이 지난 수업 |
| **진행 중** | 현재 진행 중인 수업 |
| **예정** | 아직 시작 전인 수업 |

---

## 4. 활용 팁

- 오늘 수업을 한눈에 보고, 출석·일지 처리가 필요한 수업으로 바로 이동할 수 있습니다.
- 자주 쓰는 메뉴는 대시보드 카드로 빠르게 접근합니다.
