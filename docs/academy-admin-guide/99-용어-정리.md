# 99. 용어 정리

## 1. 수업 관련

| 용어 | 설명 |
|------|------|
| **클래스(Class)** | 수업의 종류. 예: "월요 기초발레", "금요 힙합" |
| **클래스 마스터** | 등록된 클래스 정보 (유형, 장르, 난이도, 수강권 연결 등) |
| **스케줄(Schedule)** | 특정 날짜·시간에 진행되는 수업 세션 |
| **Regular** | 정기 수업 (주 2회 등 반복) |
| **Popup** | 1회성·단기 이벤트 수업 |
| **Workshop** | 집중 워크샵·특강 |

---

## 2. 수강권 관련

| 용어 | 설명 |
|------|------|
| **기간제 수강권** | 기간 내 무제한 수강, 횟수 차감 없음 |
| **쿠폰제(횟수제) 수강권** | N회권, 사용 시 1회 차감 |
| **워크샵(특강) 수강권** | 특정 워크샵 수업 전용 N회권 |
| **user_tickets** | 회원이 보유한 수강권 (구매 시 생성) |
| **remaining_count** | 잔여 횟수 (기간제는 null = 무제한) |
| **expiry_date** | 만료일 |

---

## 3. 예약·출석

| 용어 | 설명 |
|------|------|
| **신청(Enrollment)** | 수업 예약 |
| **출석** | 실제 참석 여부 |
| **결강** | 신청했으나 불참 |
| **게스트** | 비회원으로 신청한 경우 |
| **CONFIRMED** | 확정 |
| **PENDING** | 대기 |
| **CANCELLED** | 취소 |

---

## 4. 기타

| 용어 | 설명 |
|------|------|
| **academy_students** | 학원에 등록된 회원 |
| **revenue_transactions** | 결제·거래 기록 |
| **ticket_classes** | 수강권-클래스 연결 (어떤 수강권으로 어떤 클래스를 들을 수 있는지) |
