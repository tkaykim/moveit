# 10. 강사 관리

## 1. 개요

학원 소속 강사를 등록·수정하고, 클래스에 배정합니다.

**경로**: `/academy-admin/{학원ID}/instructors`

---

## 2. 강사 등록

1. **강사 등록** 버튼 클릭
2. **이름(한글/영문)**, **연락처**, **이메일** 등 입력
3. **저장**

- 기존 사용자(회원)를 검색해 강사로 등록할 수도 있습니다.

---

## 3. 강사 목록

- **이름**, **연락처**, **담당 클래스** 등 표시
- **편집**: 클릭 시 수정 모달
- **삭제**: 강사 계정 비활성화 (이미 배정된 클래스는 유지될 수 있음)

---

## 4. 클래스 배정

- **클래스(반) 관리**에서 클래스를 만들 때 **강사**를 선택합니다.
- 한 강사가 여러 클래스를 담당할 수 있습니다.
- **스케줄 관리**에서 반복 일정을 만들 때도 강사를 지정합니다.
