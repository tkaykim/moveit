# 07. 연장/일시정지 관리

## 1. 개요

회원이 신청한 수강권 **연장** 또는 **일시정지** 요청을 승인·거절하는 화면입니다. 관리자가 직접 연장/일시정지를 등록할 수도 있습니다.

**경로**: `/academy-admin/{학원ID}/extension-requests`

---

## 2. 신청 목록

### 2-1. 필터

| 필터 | 설명 |
|------|------|
| **전체** | 모든 신청 |
| **대기** | 승인/거절 대기 중 |
| **승인** | 승인된 신청 |
| **거절** | 거절된 신청 |

### 2-2. 표시 정보

- **회원** (user_tickets → user)
- **수강권명**
- **신청 유형**: 연장(EXTENSION) / 일시정지(PAUSE)
- **연장 일수** 또는 **일시정지 기간**
- **사유**
- **상태**: 대기 / 승인 / 거절

---

## 3. 승인·거절

### 3-1. 승인

1. 대기 중인 신청의 **승인** 버튼 클릭
2. 승인 시:
   - **연장**: `user_tickets.expiry_date`가 연장 일수만큼 늘어남
   - **일시정지**: absent 기간만큼 만료일이 뒤로 밀림
   - **기간제**: absent 기간 예약 취소 후, 연장된 기간에 예약 재생성

### 3-2. 거절

1. **거절** 버튼 클릭
2. **거절 사유** 필수 입력
3. 거절 처리

---

## 4. 관리자 직접 연장/일시정지

**관리자 직접 연장/일시정지 생성** 버튼 클릭:

1. **회원 검색**: 이름·연락처로 회원 검색
2. **수강권 선택**: 해당 회원의 보유 수강권 선택
3. **유형 선택**: 연장 / 일시정지
4. **연장**: 연장 일수 입력
5. **일시정지**: 시작일·종료일 입력
6. **사유** 입력
7. **등록** → 즉시 반영 (승인 단계 없음)

---

## 5. 연장/일시정지 정책 설정

- **최대 연장 일수**: 학원별로 설정 가능
- 설정 경로: **설정** → 연장/일시정지 설정 또는 본 화면 내 정책 카드
