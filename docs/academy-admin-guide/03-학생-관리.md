# 03. 학생(회원) 관리

## 1. 개요

해당 학원에 등록된 학생(회원)을 조회·검색하고, 보유 수강권과 예약 현황을 관리합니다.

**경로**: `/academy-admin/{학원ID}/students`

---

## 2. 학생 목록

### 2-1. 표시 정보

| 컬럼 | 설명 |
|------|------|
| **이름** | 회원 이름 또는 닉네임 |
| **연락처** | 전화번호 |
| **잔여 횟수** | 보유 수강권 잔여 횟수 합계 (기간제는 1로 계산) |
| **가장 가까운 만료일** | 가장 빨리 만료되는 수강권의 만료일 |
| **상태** | 수강중 / 만료예정 / 휴면 |

### 2-2. 검색

- **검색창**: 이름, 연락처, 이메일로 검색 가능
- 검색어 입력 시 실시간으로 목록이 필터링됩니다.

### 2-3. 필터

| 필터 | 설명 |
|------|------|
| **전체** | 모든 학생 |
| **수강중** | 유효한 수강권이 있고, 7일 이내 만료되는 수강권이 없는 경우 |
| **만료예정** | 7일 이내 만료되는 수강권이 있는 경우 |
| **휴면** | 유효한 수강권이 없는 경우 |

---

## 3. 학생 등록

1. **학생 등록** 버튼 클릭
2. 모달에서 **이름, 닉네임, 연락처, 이메일** 등 입력
3. **등록** 버튼 클릭

- 등록 시 해당 학원의 `academy_students`에 자동 등록됩니다.
- 수강권 구매 시에도 학생으로 자동 등록됩니다.

---

## 4. 학생 상세 (클릭 시)

목록에서 행을 클릭하면 **상세 모달**이 열립니다.

### 4-1. 표시 내용

- **인적사항**: 이름, 닉네임, 연락처, 이메일
- **보유 수강권**: 수강권별 유형(기간제/쿠폰/워크샵), 잔여 횟수, 만료일, 상태

### 4-2. 보유 수강권 표기

| 유형 | 잔여 표기 |
|------|----------|
| **기간제** | 잔여: 무제한 |
| **쿠폰제(횟수제)** | 잔여: N회 |
| **워크샵(특강)** | 잔여: N회 |

### 4-3. 수정

- **수정** 버튼으로 이름, 닉네임, 연락처, 이메일을 수정할 수 있습니다.

---

## 5. 엑셀 다운로드

- **엑셀 다운로드** 버튼을 클릭하면 현재 목록을 엑셀 파일로 내려받을 수 있습니다.
