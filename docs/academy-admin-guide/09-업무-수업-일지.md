# 09. 업무/수업 일지

## 1. 개요

날짜별 수업 일지를 작성하고, 출석·결강·동영상 링크 등을 기록합니다.

**경로**: `/academy-admin/{학원ID}/logs`

---

## 2. 화면 구성

### 2-1. 날짜 선택

- **날짜 선택**으로 특정 날짜 지정
- 해당 날짜의 수업 목록이 표시됩니다.

### 2-2. 수업 목록

- 날짜에 배정된 수업(스케줄) 단위로 목록 표시
- 수업 클릭 시 **일지 모달** 또는 **출석/신청 관리**로 이동

---

## 3. 일지 작성

### 3-1. 입력 항목

| 항목 | 설명 |
|------|------|
| **정원** | 해당 수업 최대 인원 |
| **현재 신청자 수** | 자동 반영 |
| **실제 출석자 수** | 수동 입력 |
| **동영상 URL** | 수업 녹화 링크 (선택) |
| **비고** | 메모 (선택) |

### 3-2. 출석·결강

- 신청자별로 **출석 / 결강** 표시
- 일지 저장 시 `schedules` 테이블의 `current_students`, `present_students` 등 반영

---

## 4. 활용

- 수업 후 출석률, 동영상 링크를 기록하여 나중에 참고할 수 있습니다.
- 일부 화면에서는 **오늘 수업** 섹션에서 바로 일지 작성/수정으로 이동할 수 있습니다.
