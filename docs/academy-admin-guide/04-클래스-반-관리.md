# 04. 클래스(반) 관리

## 1. 개요

수업의 종류(클래스)를 정의하는 화면입니다. Regular, Popup, Workshop 세 가지 유형을 지원합니다.

**경로**: `/academy-admin/{학원ID}/class-masters`

---

## 2. 클래스 유형

| 유형 | 용도 | 수강권 |
|------|------|--------|
| **Regular** | 정기 수업 (예: 주 2회 기초반) | 기간제 수강권 |
| **Popup** | 1회성 이벤트·단기 수업 | 쿠폰제(횟수제) 수강권 |
| **Workshop** | 집중 워크샵·특강 | 워크샵(특강) 수강권 |

---

## 3. 클래스 등록

### 3-1. 등록 흐름

1. **클래스 등록** 버튼 클릭
2. **Step 1**: 유형 선택 (Regular / Popup / Workshop)
3. **Step 2**: 기본 정보 입력
   - 수업명, 장르, 난이도, 홀, 강사 등
4. **Step 3**: 수강권 설정
   - 이 클래스를 들을 수 있는 수강권 선택

### 3-2. 기본 정보

| 항목 | 설명 |
|------|------|
| **수업명** | 클래스 표시 이름 |
| **장르** | 발레, 재즈, 힙합, 케이팝 등 |
| **난이도** | 초급 / 중급 / 고급 |
| **홀** | 수업 진행 장소 |
| **강사** | 담당 강사 |
| **포스터** | 수업 이미지 (선택) |
| **설명** | 수업 소개 (선택) |

### 3-3. 수강권 연결

- **정규(Regular)**: 기간제 수강권 선택
- **팝업(Popup)**: 쿠폰제 수강권 선택, 무료 수강 조건(정규 수강권 보유자) 설정 가능
- **워크샵(Workshop)**: 워크샵 수강권 선택

---

## 4. 필터

| 필터 | 설명 |
|------|------|
| **전체 / 활성 / 비활성** | 클래스 활성 여부 |
| **전체 / Regular / Popup / Workshop** | 클래스 유형 |

---

## 5. 클래스 카드

- **활성/비활성 토글**: 클릭으로 즉시 전환
- **연결된 수강권**: 해당 클래스에서 사용 가능한 수강권 이름 표시
- **편집**: 클릭 시 수정 모달 열림
