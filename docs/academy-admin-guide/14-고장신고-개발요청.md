# 14. 고장신고/개발요청

## 1. 개요

버그 신고, 기능 요청, 문의 사항을 제출하는 화면입니다.

**경로**: `/academy-admin/{학원ID}/support`

---

## 2. 사용 방법

1. **고장신고** 또는 **개발요청** 선택
2. **제목**, **내용** 입력
3. **제출**

- 제출된 내용은 운영팀에서 확인·처리합니다.
- 이메일 또는 앱 알림으로 회신이 올 수 있습니다 (구현 여부는 배포 환경에 따름).
