# 05. 스케줄 관리

## 1. 개요

월별 달력으로 수업 일정을 조회·등록·수정합니다. 반복 일정 생성과 일별 수업 확인이 가능합니다.

**경로**: `/academy-admin/{학원ID}/schedule`

---

## 2. 화면 구성

### 2-1. 상단

- **스케줄 생성**: 반복 일정 등록 모달 열기
- **검색**: 수업명, 강사명, 장르, 홀명으로 필터
- **홀 필터**: 특정 홀만 보기 (전체 / 홀별)

### 2-2. 난이도 범례

| 색상 | 난이도 |
|------|--------|
| 초록 | 초급 |
| 노랑 | 중급 |
| 빨강 | 고급 |

### 2-3. 달력

- **월 이동**: ◀ ▶ 버튼으로 이전/다음 달 이동
- **날짜 셀**: 해당 날짜의 수업 목록 표시
- **오늘**: 파란색 배경으로 구분
- **클릭**: 날짜 클릭 시 **일별 수업 모달** 열림

---

## 3. 스케줄 생성 (반복 일정)

### 3-1. 진입

**스케줄 생성** 버튼 → 유형 선택 (Regular / Popup / Workshop)

### 3-2. Regular (정규 수업)

1. **클래스 선택**
2. **수강권 설정**: 이 수업에 사용 가능한 수강권 선택
3. **요일·시간·홀 설정**
4. **기간 설정**: 적용할 시작일·종료일
5. **생성** → 지정된 날짜에 스케줄 자동 생성

### 3-3. Popup (팝업 수업)

1. **클래스 선택** (Popup 유형만)
2. **무료 수강 조건**: 특정 정규 클래스 기간제 수강권 보유자 무료 수강 여부
3. **요일·시간·홀·날짜** 설정
4. **생성**

### 3-4. Workshop (워크샵)

1. **클래스 선택** (Workshop 유형만)
2. **수강권 설정**
3. **날짜·시간·홀** 설정
4. **생성**

---

## 4. 일별 수업 모달

날짜 셀을 클릭하면 해당 날짜의 수업 목록이 모달로 표시됩니다.

- **수업 추가**: 해당 날짜에 새 수업 등록
- **수업 수정/삭제**: 개별 수업 편집
- **신청인원**: 출석/신청 관리로 이동

---

## 5. 활용 팁

- **검색**으로 특정 수업·강사·홀만 모아서 확인할 수 있습니다.
- **홀 필터**로 장소별 수업을 구분해 볼 수 있습니다.
- 반복 일정은 **스케줄 생성**으로 한 번에 여러 날짜에 등록할 수 있습니다.
