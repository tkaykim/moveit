# 결제 수단(카드) 보안 안내

## Supabase에 저장되는 데이터

**카드 번호·CVC·유효기간 등 결제 정보는 저장하지 않습니다.**

저장하는 항목은 아래만입니다.

| 항목 | 설명 |
|------|------|
| `toss_billing_key` | 토스페이먼츠가 발급한 **빌링 키(토큰)**. 자동결제 요청 시 이 토큰으로 토스 API를 호출하며, 카드 번호가 아님. |
| `toss_customer_key` | 우리가 만든 학원별 고객 식별자 (예: `academy_xxx`) |
| `card_company` | 카드사 이름 (예: "신한", "현대") |
| `card_number_masked` | **마스킹된 번호만** (예: `****1234`). 토스 API가 인증 완료 후 넘겨주는 값 그대로 저장. |

실제 카드 번호는 토스페이먼츠 측에서만 처리하며, 우리 서버·DB에는 전혀 들어오지 않습니다.

## PCI-DSS

카드 번호를 저장·처리하지 않으므로, 우리 시스템은 PCI-DSS 적용 범위에서 카드 데이터를 다루는 영역에 해당하지 않습니다. 자동결제는 모두 토스페이먼츠 API(빌링 키)를 통해 이루어집니다.
